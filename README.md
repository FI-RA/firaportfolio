# ** Evaluación modular M2 **

## ** Resumen **
Este es un proyecto web que contempla una landing page construida con 
HTML, CSS, JS y Bootstrap v5. El objetivo fue crear un portafolio web que se alojase
en un repositorio en GitHub y utilizar GitHub Pages para compartirlo con una URL.

### Estructura proyecto

Consta de una carpeta assets donde se guardan los archivos de estilo dentro de la carpeta **css**
y los archivos estáticos como imágenes dentro de la carpeta **img**.
También el archivo **index.html** el cual contiene todo el código estructural de la pógina web.
Además se emplea Bootstrap utilizando el CDN.

Posee una sección Hero con animaciones hechas con CSS puro, íconos de de Linkedin, Whatsapp y GitHub, 
una sección de estilo Bento Grid donde se divide en cuatro contenedores donde se muestra una
fotografía con una descripción profesional, educación, habilidades técnicas y blandas y un
formulario de contacto con validaciones en tiempo real. Este último solo cumple una función estética.
